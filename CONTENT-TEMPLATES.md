## 我们是谁

ChainTrace 是一款专业的区块链资金流向可视化分析平台，致力于为区块链安全研究人员、执法机构、交易所和 DeFi 项目提供强大的链上追踪工具。

## 核心优势

### 🌐 多链支持
支持 Solana、Ethereum 等主流区块链，未来将扩展到更多公链。

### 📊 可视化图谱

快速查询链上交易，实时更新资金动向。

- **区块链**: RPC 节点 + 智能合约解析

### 3. 使用文档页面 (/docs)

```markdown
# 使用文档

## 快速开始

### 第一步：输入地址
在搜索框中输入要追踪的区块链地址：
- Solana 地址：44 个字符
- Ethereum 地址：0x 开头，42 个字符

### 第二步：选择链
选择对应的区块链网络：
- Solana Mainnet
- Ethereum Mainnet
- 更多链即将支持

### 第三步：查看结果
系统将自动展示：
- 交易图谱
- 交易列表
- 风险评估
- 统计数据

## 功能详解

### 交互操作

**图谱操作：**
- 🖱️ 拖拽：移动节点位置
- 🔍 滚轮：缩放图谱
- 👆 点击：查看节点详情
- 🔄 双击：展开子交易

**筛选功能：**
- 只看转入
- 只看转出
- 按金额排序
- 按时间排序

### 风险标识

**颜色含义：**
- 🟢 绿色：原始查询地址
- 🔴 红色：高风险地址
- 🟡 黄色：中等风险
- ⚪ 灰色：普通地址

**风险类型：**
- Phishing：钓鱼诈骗
- Hacking：黑客攻击
- Mixer：混币器
- Scam：诈骗项目

### 数据导出

**CSV 格式：**
```csv
address,type,amount,tx_hash,timestamp
0x123...,in,100,0xabc...,2026-01-08
```

**JSON 格式：**
```json
{
  "address": "0x123...",
  "transactions": [...],
  "risk_score": 0.8
}
```

## 常见问题

### Q: 支持哪些区块链？
A: 目前支持 Solana 和 Ethereum，未来将支持 BSC、Polygon 等。

### Q: 数据更新频率？
A: 实时查询链上数据，延迟 < 1 分钟。

### Q: 如何判断地址风险？
A: 基于多个威胁情报数据库和 AI 分析模型。

### Q: 可以追踪多少层？
A: 默认 3 层，最多支持 10 层深度追踪。

### Q: 数据是否保存？
A: 查询结果缓存 24 小时，不会永久保存用户数据。

## API 文档

### 查询地址交易
```bash
GET /api/address/{address}/transactions
```

### 获取交易图谱
```bash
GET /api/address/{address}/graph?depth=3
```

### 风险检查
```bash
GET /api/risk/check?address={address}
```

---

**需要帮助？联系我们：tech@matrixlab.work**
```

## 🎯 SEO 优化要点

### 每篇内容必须包含：

1. **标题优化**
   - 包含核心关键词
   - 60 字符以内
   - 吸引点击

2. **描述优化**
   - 总结文章要点
   - 160 字符以内
   - 包含关键词

3. **内部链接**
   - 链接到相关文章
   - 链接到产品页面
   - 自然融入内容

4. **外部链接**
   - 引用权威来源
   - 添加 rel="nofollow" 给不信任的链接

5. **图片优化**
   - 添加 alt 文本
   - 压缩文件大小
   - 使用描述性文件名

6. **结构化**
   - 使用 H1-H6 标题
   - 段落简短
   - 使用列表和表格

7. **CTA（行动号召）**
   - 文章结尾引导
   - 明确的下一步
   - 链接到产品
